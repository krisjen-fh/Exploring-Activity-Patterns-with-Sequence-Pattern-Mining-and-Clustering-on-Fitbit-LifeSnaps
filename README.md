# Exploring-Activity-Patterns-with-Sequence-Pattern-Mining-and-Clustering-on-Fitbit-LifeSnaps
Proyek Eksplorasi Pola Aktivitas-Tidur dengan Sequence Pattern Mining dan Clustering pada Fitbit LifeSnaps adalah proyek akhir dari mata kuliah Data Mining. Proyek ini bertujuan untuk menganalisis pola aktivitas harian pengguna perangkat FitBit menggunakan metode Sequence Pattern Mining dan Clustering. Data yang digunakan mencakup data harian, per jam, dan hasil survei psikologis dari dataset FitBit LifeSnaps. Analisis dimulai dengan Exploratory Data Analysis (EDA) dan pre-processing untuk menangani data hilang, outlier, serta transformasi fitur.

Pada tahap Sequence Pattern Mining, algoritma PrefixSpan digunakan untuk mengidentifikasi pola aktivitas paling sering, yang kemudian digunakan untuk mengelompokkan pengguna menjadi tiga segmen: Aktif & Sehat, Kurang Gerak, dan Stres/Kurang Istirahat. Sementara itu, analisis clustering dilakukan menggunakan fitur-fitur fisiologis dan aktivitas, seperti jumlah langkah, detak jantung istirahat, BMI, dan waktu sedentari, untuk mengidentifikasi kelompok risiko kesehatan.

Selanjutnya, data hasil survei psikologis seperti STAI, BREQ, TTM, PANAS, dan Personality digabungkan dengan data aktivitas untuk memahami hubungan antara pola hidup dengan stres, motivasi olahraga, dan kepribadian. Hasilnya menunjukkan bahwa gaya hidup aktif berkorelasi dengan tingkat stres lebih rendah dan motivasi olahraga yang bersifat intrinsik.

Proyek ini memberikan wawasan mendalam tentang hubungan antara aktivitas fisik, kondisi emosional, dan profil psikologis pengguna, serta menunjukkan potensi intervensi berbasis data untuk meningkatkan gaya hidup sehat.
